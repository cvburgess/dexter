# Changelog

## v2.0.0

- Dexter is now a single, universal app for iOS, Android, and the web — one planner, everywhere you work
- Plan your day with prioritization, markdown notes, a customizable journal, habit and goal tracking, and calendar feeds
- Set native alarms on tasks so nothing slips
- Connect your favorite AI tools like Claude, ChatGPT, and Gemini to plan alongside you
- Dexter is now open source under the MIT license
