# Dexter app

Expo (React Native) + Expo Router. Runs on **iOS**, **Android**, and **web**.

## Setup

```bash
npm install --legacy-peer-deps
```

Use `--legacy-peer-deps` if npm reports peer dependency conflicts (common with React 19 + test tooling).

## Commands

| Script                            | Description             |
| --------------------------------- | ----------------------- |
| `npm start`                       | Dev server              |
| `npm run web`                     | Web only                |
| `npm run ios` / `npm run android` | Open simulator / device |
| `npm run lint`                    | ESLint                  |
| `npm run format`                  | Prettier                |
| `npm test`                        | Jest                    |

See [`../docs/frontend.md`](../docs/frontend.md) for conventions.
